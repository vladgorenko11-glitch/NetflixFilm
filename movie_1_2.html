Поделитесь этой ссылкой — другие люди увидят страницу со всеми вашими изменениями, потому что данные сохраняются прямо в GitHub-репозиторий сайта.

⚙ Панель администратора
✎ Совет: в режиме редактирования просто кликните на любой текст на странице (заголовок, бейджи, описание и т.д.) и начните печатать — изменения автоматически сохраняются в GitHub через несколько секунд.
🔗 Хранилище (GitHub)
Настройте подключение к репозиторию на главной странице (кнопка «⚙ GitHub»), чтобы изменения сохранялись для всех.
📝 Текст сайта (шапка и подвал)
Этот текст общий для всего сайта — он отобразится и в каталоге, и на всех страницах фильмов.
▶ Видео
Сайт не хранит сами видеофайлы — вставьте прямую ссылку на видео (с Google Drive, другого хостинга и т.п.). Ссылка сохраняется в GitHub вместе с остальными данными фильма.
🖼 Фон сайта (хиро-баннер)
🖱 Перетаскивайте фото мышью, чтобы сдвинуть. Колесо мыши — приближение/отдаление. Нажмите кнопку выше ещё раз, чтобы выйти из режима.
🎨 Цвет фона сайта
Выберите 2 цвета для фона — они смешиваются градиентом по всему сайту (от чёрного и белого до любых оттенков).
