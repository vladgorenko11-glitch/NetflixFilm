Настройки хранилища (GitHub)
Чтобы все посетители видели одну и ту же библиотеку, сайт сохраняет данные фильмов прямо в ваш GitHub-репозиторий (тот же, где лежит сайт на GitHub Pages). Для записи нужен личный токен доступа.
Репозиторий
Токен доступа
Создайте токен на странице github.com/settings/tokens (Fine-grained token) → выберите ваш репозиторий → раздел Permissions → Contents → Read and write. Скопируйте токен сюда. Токен хранится только в этом браузере и нигде не публикуется.
